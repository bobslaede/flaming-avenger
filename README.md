# flaming-avenger

## install

```bash
$ npm install -g gulp bower
$ npm install -d
$ bower install
$ node --harmony ./bower-polymer.js
```

## run dev

run `src/index.html` in a browser

## build and run

```bash
$ gulp build:html
$ gulp build:elements
$ gulp build:css
$ gulp build:js
```

run `dist/index.html` in a browser
